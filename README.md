Annotador POO - Frontend
=======================

Este é um projeto frontend standalone feito para um exercício de Programação Orientada a Objetos (POO).
Ele implementa classes em JavaScript (Item, Task, Bill, Annotator) e usa localStorage para persistência.

Como usar:
1. Extraia os arquivos.
2. Abra o arquivo index.html em um navegador moderno.
3. Adicione contas ou tarefas, marque como concluído/pago, exporte ou importe um arquivo JSON.

Arquivos:
- index.html
- styles.css
- app.js
- README.md

Desenvolvido como um frontend simples e funcional para seu projeto pessoal.
